# HR Headcount Dashboard

## Project Overview
This Power BI mini project analyzes employee headcount data to track workforce distribution, department-wise employee count, and HR trends.

## Tools Used
- Power BI
- Power Query
- DAX

## Features
- Department-wise employee analysis
- Employee headcount tracking
- Workforce distribution insights
- Interactive filtering and drill-down

## Key KPIs
- Total Employees
- Department Count
- Gender Distribution
- Workforce Trends

## Dashboard Insights
- Identify departments with higher employee count
- Analyze workforce distribution
- Support HR decision making
